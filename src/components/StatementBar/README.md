# Statement Bar

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.StatementBar />
```

## Props

- message: [string], message inside of the component
- sectionClassName: [string], manual style to the section box
- messageClassName: [string], manual style for the text
