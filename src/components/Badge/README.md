# Badge

Simple badge label. It has a default display and comes with some preset classNames for danger, warning, info, success variations.

## Props

- title: This is used to provide hover and screen reader contextual information
- className: This is used to provide either special classNames to be applied or use one of the presets of `.badge--danger`, `.badge--warning`, `.badge--success`, `.badge--info`.

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.Badge title="My Title" className="myClassName">Some children</Badge>
```
