# Breadcrumbs

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.Breadcrumbs />
```

## Props

- trails: [array], an array of string values used as titles for the breadcrumbs
- links: [array], an array of string values used as links in relations to the trails array
- className: [string], class that could be added to the trails for own custom ability
