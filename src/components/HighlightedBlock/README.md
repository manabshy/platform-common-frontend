# Highlighted Block

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.HighlightedBlock />
```

## Props

- backgroundColour: [string], examples -> "#edf7ff" or "red"
- borderLeftColour: [string], examples -> "#edf7ff" or "red"
- blockClassName: [string], Add your own class for customisation

## Children

- {children}: Anything can be passed into this component to render inside the highlighted block.

```javascript
<HighlightedBlock>
    <span>Anything</span>
</HighlightedBlock>
```
