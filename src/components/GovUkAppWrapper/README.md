# GovUk App Wrapper

A wrapper component which provides renders the GovUkTemplate component with a FeedbackBanner and page content.

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.GovUkAppWrapper />
```
