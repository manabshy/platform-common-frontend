# GovUk Footer

GovUk style footer with options to add a set of links. Auto includes the NCSC Ts & Cs link and crown logo.

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.GovUkFooter />
```
