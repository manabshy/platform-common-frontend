# GDS Input

Text input field component.

## Usage

```javascript
import { Components as PCFComponents } from 'platform-common-frontend'

<PCFComponents.GDSInput />
```

GDS style text input. Allows

- labels
- hints (see GDS docs)
- error message
- autofocus
- change/blur events
- textarea / text field (default)
- max characters with remaining counter
